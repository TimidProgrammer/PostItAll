Using CSS class in new notes

This is a mixed example with overwriten css clases and custom clases passed as note parameters. All css clases are defined and included from the noteconfiguration.css stylesheet file.

Note with $.fn.postitall.cssclases.note class.
Without class the style will be defined by plugin properties Note with .customNote class (changed in note config) Note with .note1 class (passed in the new call) Note with .tv class Note with an inexistent class or without class.
