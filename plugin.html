Post It All! jQuery plugin

A jQuery plugin to add postits to your webpage

Basic example

This is a basic example, where you can convert the content of an HTML object into an editable, draggable and resizable postit.

Code

// HTML
<div id="idAddDemoPostit">Content of the postit 1</div>

//JAVASCRIPT
<script>
$(document).ready(function() {
  $('#idAddDemoPostit').postitall();
});
</script>

Form example

In this example, we pass the parameters of the form to create a new postit. With the 'newPostit' parameter set to true, you can create as many post it's as you want.

Code

// HTML
<form>
  <label for="idBgTextColor">Background-Color:</label> 
  <input id="idBgTextColor" type="text" value="#0000ff">
  <label for="idTextColor">Color:</label> 
  <input id="idTextColor" type="text" value="#333333">
  <label for="idContent">Content:</label> 
  <textarea id="idContent">This is anoteher demo</textarea>
  <p><input type="button" class="btn btn-primary" value="Add this as a postit" id="idAddDemoPostit2"></p>
</form>

// JAVASCRIPT
<script>
$(document).ready(function() {
  $('#idAddDemoPostit2').click(function() {
      $('#idRight').postitall({
        'textcolor'       : $('#idTextColor').val(),
        'backgroundcolor' : $('#idBgTextColor').val(),
        'description'     : $('#idContent').val(),
        'newPostit'       : true
      });
  });
});
</script>
