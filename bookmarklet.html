Post It All!

Add a post it to any webpage, and recover it later with Post it all!
Just add the bookmarklet in your bookmark toolbar and start "Postit-ing" all!!


Post It All!

It's so easy

  1. Install it by adding the bookmarklet to your bookmarks toolbar.
  2. Press the "Post It All!" bookmark
  3. Add post it's to your page pressing the "Add postit" button at the left of the window
  4. Recover it later by pressing the "Post It All!" bookmark

No login required

This service is anonymously and all the stored data is only stored in your local web browser. This is an advantage because all the data you store, is only accessible by you, locally in this web browser / machine, and a disadvantage because this is not a cloud storage service, so, your data will be not recoverable by other browsers or machines.

Remember anything

You can add as many post it's as you want. You can write annotations, remembers, to do's list, or whatever you imagine in any web page you visit. Later, you can recover all the stuff when you needed, only by pressing the "Post It All!" bookmarklet.

About Post It All. What, who, how and why.

"Post It All!", is just another sticky notes javascript plugin for your favorite web pages.
It saves data throw HTML5 Web Storage locally. So, all the data you store, is only accesible in this machine and in this browser.

It was developed in my spare time just for fun. You can check my entery profile here: Javi Filella

Current Features

  • Drag & Resize every sticky
  • Personalize every postit with diferent background colors, text color and text shadow.
  • Create a postit with the "Add postit" button in the middle-left side of the page (try to select some text before)
  • Create a postit in your mouse position by pressing: "Caps + mouse click" (try to select some text before)

Limits & problems

  • Post It All will not work in old browsers. Also, ensure your browser have support to the HTML5 Web Storage technology
  • In Chrome browser, the Content Security Policy restrict the use of the bookmarklet in some pages like google. I'm currently working in the "Post It All! Chrome Extension", that will not be limited, and will be autoloaded in the begining of each page.
  • Note that the content note will try to resize the postit while you type in. In some cases, this could result in a strange behaviour, when you paste html content with float content. If the note content go out of the postit, try to resize it, or add an space at the end of the note to forze the resize.
